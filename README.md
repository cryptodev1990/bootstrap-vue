## Quick Start

Install via **NPM**:   

```npm i --save-dev bootstrap-vue```

Install via **YARN** (recommended):   

```yarn add --dev bootstrap-vue```


```js
import Vue from 'vue'
import BootstrapVue from 'bootstrap-vue';

// Globally register bootstrap-vue components
Vue.use(BootstrapVue);
````
